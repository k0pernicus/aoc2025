# AOC 2025

Attempt to solve advent of code 2025 exercises.

Usage of Python version 3.13.7.

## How to use

```sh
python main.py <EXERCISE_MODULE> <INPUT_FILE>
```

For example, if you want to solve the first exercise with an example:

```sh
python main.py ex01 examples/ex01_01.txt
```

You can enable the debug logs using the `--debug` flag.
